# Surge
A quick, shade-able api for making bukkit plugins quickly

### Maven Coordinates
``` xml
<repository>
   <id>surge</id>
   <url>https://raw.githubusercontent.com/cyberpwnn/Surge/master/repository/</url>
</repository>

<dependency>
  <groupId>surge</groupId>
  <artifactId>Surge</artifactId>
  <version>1.0</version>
  <scope>provided</scope>
</dependency>
```
